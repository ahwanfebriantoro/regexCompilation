### Nomor telp
```/^((\\+[1-9]{1,4}[ \\-]*)|(\\([0-9]{2,3}\\)[ \\-]*)|([0-9]{2,4})[ \\-]*)*?[0-9]{3,4}?[ \\-]*[0-9]{3,4}?$/```
<hr />

### Nomor telp indonesia
```/(\+62 ((\d{3}([ -]\d{3,})([- ]\d{4,})?)|(\d+)))|(\(\d+\) \d+)|\d{3}( \d+)+|(\d+[ -]\d+)|\d+/```

### Username
```/^[a-z0-9_-]{3,16}$/```
<hr />

### Email
```/^([a-z0-9_\.-]+)@([\da-z\.-]+)\.([a-z\.]{2,6})$/```
<hr />

### Password
```/^[a-z0-9_-]{6,18}$/```
<hr />

### Hex Number
```/^#?([a-f0-9]{6}|[a-f0-9]{3})$/```
<hr />

### Slug(pretty url)
```/^[a-z0-9-]+$/```
<hr />

### IP Address
```/^(?:(?:25[0-5]|2[0-4][0-9]|[01]?[0-9][0-9]?)\.){3}(?:25[0-5]|2[0-4][0-9]|[01]?[0-9][0-9]?)$/```
<hr />

### HTML tag
```/^<([a-z]+)([^<]+)*(?:>(.*)<\/\1>|\s+\/>)$/```
